# 🧠 AI Resume Analyzer

A web application that analyzes resumes and provides insights using AI.

---

## 🚀 Features

- Upload resume (PDF/DOCX)
- Extract key information
- Analyze resume using AI
- Display results on dashboard

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML, CSS
- MySQL
- PyPDF2, python-docx

---

## 📂 Project Structure

```
Resume-Analyser/
│
├── app.py
├── models.py
├── db.py
├── ai.py
├── templates/

```

---

## ▶️ How to Run

```bash
git clone https://github.com/AkshitaVermaa/Resume-Analyser.git
cd Resume-Analyser
pip install -r requirements.txt
python app.py
```
